# Command Line Diagnostics Toolkit

This repository contains command line tools and scripts I created while learning system diagnostics and troubleshooting.

The goal is to build a toolkit of simple command line utilities that help quickly assess the health of a system.

## Focus Areas

- System health checks
- Network diagnostics
- Disk analysis
- Process monitoring
- Troubleshooting automation

## Current Tools

### KyleDiagnostic.ps1

A Windows PowerShell script that gathers important system information for troubleshooting.

Checks include:

- system information
- disk usage
- network configuration
- running processes

## Goal of This Project

Create a growing collection of command line diagnostic tools that can quickly evaluate system health and help identify issues.
